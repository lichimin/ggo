# 玩家属性汇总表

| 英文标识 | 中文标识 | 效果说明 |
|---------|---------|---------|
| hp | 生命值 | 增加角色的最大生命值 |
| attack | 攻击力 | 增加角色的基础攻击力 |
| attack_speed | 攻击速度 | 增加角色的攻击频率 |
| move_speed | 移动速度 | 增加角色的移动速度 |
| bullet_speed | 子弹速度 | 增加角色发射子弹的飞行速度 |
| drain | 吸血 | 攻击时吸取敌方生命值的百分比 |
| critical | 暴击值 | 影响攻击暴击的概率（内部计算使用） |
| critical_rate | 暴击率 | 增加攻击暴击的概率 |
| critical_damage | 暴击伤害 | 增加暴击时的伤害倍率 |
| dodge | 闪避 | 减少被敌方攻击命中的概率 |
| instant_kill | 秒杀 | 攻击时有概率直接秒杀敌人 |
| recovery | 恢复 | 每秒自动恢复的生命值 |
| trajectory | 弹道 | 增加角色一次发射的子弹数量 |
| damage_reduction | 减伤 | 减少受到的伤害百分比 |
| atk_type | 攻击类型 | 改变攻击的类型或效果 |
| enhance | 强化加成 | 增加装备的总体属性效果（如七宗罪属性） |

    ## 攻击类型说明

    | 类型值 | 中文名称 | 效果说明 |
    |-------|---------|---------|
    | 0 | 默认 | 默认攻击类型 |
    | 1 | 反弹 | 攻击可以反弹到其他敌人 |
    | 2 | 穿透 | 攻击可以穿透敌人 |
    | 3 | 爆炸 | 攻击有爆炸效果，对范围敌人造成伤害 |

## 特殊属性说明

### 七宗罪属性（enhance类型）
- **暴食**：增加攻击速度
- **贪婪**：增加秒杀几率
- **傲慢**：增加最大HP
- **嫉妒**：增加暴击伤害
- **色欲**：增加生命恢复
- **暴怒**：增加暴击率
- **怠惰**：增加攻击力

## 属性计算说明

1. **基础属性**：由角色本身提供基本值
2. **装备属性**：包括装备模板的基础属性和装备的附加属性
3. **皮肤属性**：由已激活的皮肤提供

所有属性会进行累加计算，最终形成玩家的总属性值。

## 属性显示格式

- **移动速度**、**子弹速度**、**吸血**、**闪避**、**秒杀**、**暴击率**：显示为百分比
- **弹道**：显示为实际数量（基础值+1）
- **暴击伤害**：显示为百分比（原倍数×100）
- **攻击类型**：显示为中文名称
- **减伤**：显示为百分比
